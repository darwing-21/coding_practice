# **Week 1: Exercise Documentation**

## **Day 1**

1. **Reverse a String**  
   **Class Name:** `ReverseString`  
   **Description:** Write a function that receives a string and returns the reversed string.  
   **Input Example:** `"hello"`  
   **Output Example:** `"olleh"`

2. **Sum of Even Numbers in a Range**  
   **Class Name:** `SumEvenNumbersInRange`  
   **Description:** Write a function that receives two numbers and returns the sum of all even numbers between them.  
   **Input Example:** `1, 10`  
   **Output Example:** `30`

3. **FizzBuzz**  
   **Class Name:** `FizzBuzz`  
   **Description:** Write a function that receives a number and returns a list of strings according to the FizzBuzz rules.  
   **Input Example:** `15`  
   **Output Example:** `["1", "2", "Fizz", "4", "Buzz", "Fizz", "7", "8", "Fizz", "Buzz", "11", "Fizz", "13", "14", "FizzBuzz"]`

4. **Prime Numbers**  
   **Class Name:** `PrimeNumberChecker`  
   **Description:** Write a function that checks if a number is prime.  
   **Input Example:** `7`  
   **Output Example:** `True`
